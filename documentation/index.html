    Builder is a HTML template for construction business companies for online presence of their business. Builder is a clean responsive HTML5 CSS3 template theme with two homepage layouts. It includes premium wordpress plugins Revolution Slider ( worth $16 ). The template comes with Projects and Services page layouts to showcase your work. Builder is the perfect one who are looking for a minimal and professional looking WordPress themes. Looking fantastic on retina devices, with beautiful typography and those small details that really make you stand out from the crowd.

How to chane color schemes?

We build awesome custom color schemes for Builder HTML template. All of versions comes custom colors for example we used red for default version. Open custom.css and if you can change all these colors to the your custom color, all red colors will be change.

How to edit jQuery Files?

We used some great jQuery plugins for this template. All our plugins has been added from jquery.custom.js. If you want to edit some single pages or just you want to edit jquery.custom.js please open it a text editor like Dreamweaver, Notepad or Notepad++ and edit any lines what you want.

How to edit HTML Files?

We have included 16 custom HTML templates like homepage styles, blog styles, gallery styles etc. Please open any HTML files with a text editor like Dreamweaver, Notepad or Notepad++ and edit any lines what you want.

How to edit CSS Files?

We have included some custom CSS styles like style.css (all site codes) etc. Please open any CSS files with a text editor like Dreamweaver, Notepad or Notepad++ and edit any lines what you want.

Files & Sources - #back to top


Included Stylesheets

These are the primary CSS files used for general front-end styling. Use these to customize your theme even further. All included JavaScript codes under Builder/css/

  • 1. style.css - Primary stylesheet
  • 2. bootstrap.css - Bootstrap stylesheet
  • 3. owl-carousel.css - OWL Carousel
  • 4. fontawesome.css - FontAwesome Font Icons stylesheet
  • 5. custom.css - Builder Color Schemes stylesheet
  • 6. Builderfonts.com - Builder used fonts
  • 7. prettyPhoto.css - Lightbox effect css file
Included JavaScript

These are the various attribution inks to the Javascript files included or modified to work with in this theme. All included JavaScript codes under Builder/js/

  • 1. bootstrap.js - Bootstrap JavaScript
  • 2. custom.js - All JavaScript Plugins
  • 3. retina.js - Retina JavaScript
  • 4. jquery.js - Base JavaScript
  • 5. prettyPhoto.js - Lightbox JavaScript
  • 6. owl-carousel.js - Lightbox JavaScript
  • 7. revslider.js - Revolution Slider JavaScript
  • 8. circle.js - Coming soon JavaScript
  • 9. progress.js - Skills bar JavaScript
